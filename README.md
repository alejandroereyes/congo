#Congo
  Congo is a marketplace for users to purchase amazing products online.

#How to use
  Go to `https://congo-place.herokuapp.com` and sign up.
  Navigate site using links.
  Add items to your cart and checkout using checkout button.

##Details
  * Ruby 2.2.0
  * Dependancies
   * gem 'bootstrap-sass'
   * gem 'react-rails'
   * gem 'bcrypt'
   * gem 'mandrill_dm'
   * gem 'faker'
